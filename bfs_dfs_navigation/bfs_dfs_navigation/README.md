# Smart Navigation System (BFS & DFS)

## 📌 Problem Description
This project finds a path between two nodes in a graph using AI search algorithms.

## 🧠 Algorithms Used
### Breadth-First Search (BFS)
- Uses Queue
- Finds shortest path

### Depth-First Search (DFS)
- Uses Recursion
- Finds any valid path

## ⚙️ Working
- Graph is predefined
- User inputs start and goal node
- BFS and DFS find paths

## 🚀 How to Run
1. Run the program:
   python main.py
2. Enter start node
3. Enter goal node

## 📊 Output
- BFS Path (shortest)
- DFS Path (not always shortest)

## 📌 Example
Input:
Start: A  
Goal: F  

Output:
BFS Path: A → B → D → F  
DFS Path: A → C → D → F
